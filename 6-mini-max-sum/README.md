# Mini / Max Sum

Return the minimum & maximum sum from an array


## Example

Given that array $b = [1, 3, 5, 7, 9];

The minimum sum is 1 + 3 + 5 + 7 = 16

The maximum sum is 3 + 5 + 7 + 9 = 24

You'd return: 16 24


## Constraints

1 <= $arr[i] <= 10e9


## Output Format

Expects a string with minimum and maximum values.
